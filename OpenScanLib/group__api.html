◆ OSc_SetDeviceModuleSearchPaths()

void OSc_SetDeviceModuleSearchPaths ( char **  paths)

Set the search path for device modules.

This function must be called before either OSc_GetAllDevices() or OSc_GetNumberOfAvailableDevices() is called.

OpenScanLib makes a copy of the path strings, so they can be released after the call.

Parameters
